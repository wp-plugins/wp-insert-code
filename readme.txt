== Description ==



This simple plugin makes it very easy to add scrips and code to your header, body (before and after 
post/page and footer).



== Installation ==



1. Upload `wp-insert-code` to the `/wp-content/plugins/` directory


2. Activate the plugin through the 'Plugins' menu in WordPress


3. Add the code and/or scripts in the fields that fits your needs.
